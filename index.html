The Atom

The Neuron

The basic unit of learned intelligence

Start simple.

The oldest equation you know.

$$y = mx + c$$

A line. Input → multiply → add → output.

$$y = wx + b$$

Rename it. w = weight. b = bias.

That's a neuron.

That's it.

That's a neuron.

But what's learning?

Step 1

Make a prediction

Forward pass — run the input through

Step 2

Measure the error

How wrong were we? → Loss function

Step 2
$$\mathcal{L} = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$$

Mean squared error — the simplest loss

Step 3

Compute gradients

Which direction reduces the error?

Step 3
$$\frac{\partial \mathcal{L}}{\partial w}$$

The gradient — the slope of the loss landscape

Step 4

Update the weights

Step 4 — Gradient Descent
$$w \leftarrow w - \eta \frac{\partial \mathcal{L}}{\partial w}$$

η = learning rate — how big a step we take

Step 5

Repeat.

Thousands. Millions. Billions of times.

This is all of machine learning.

Predict → Error → Gradient → Update → Repeat

But y = wx + b is a line.

The world isn't linear.

Non-linearity

ReLU
$$f(x) = \max(0, x)$$

Simpler. Faster. Works better.

Sub-problem 4 — The Core

Self-Attention

"What should I
pay attention to?"

Three projections from every token

$$Q = XW^Q \qquad K = XW^K \qquad V = XW^V$$

Query, Key, Value — learned linear projections

ProjectionIntuition
Query"What am I looking for?"
Key"What do I contain?"
Value"What do I deliver?"
The Attention Equation
$$\text{Attention}(Q,K,V) = \text{softmax}\!\left(\frac{QK^\top}{\sqrt{d_k}}\right)V$$
Step by step

1. Similarity

$$QK^\top$$

How much does each token relate?

2. Scale

$$\frac{QK^\top}{\sqrt{d_k}}$$

Prevent values exploding

3. Normalize

$$\text{softmax}\!\left(\frac{QK^\top}{\sqrt{d_k}}\right)$$

Convert to attention weights

4. Aggregate

$$\text{softmax}\!\left(\frac{QK^\top}{\sqrt{d_k}}\right)V$$

Weighted sum → output

Attention heatmap — "The cat sat on the mat"

Going Wider

Multi-Head Attention

Multiple attention heads
in parallel

Each head learns a different type of relationship

$$\text{head}_i = \text{Attention}(QW_i^Q, KW_i^K, VW_i^V)$$

$$\text{MultiHead} = \text{Concat}(\text{head}_1, \ldots, \text{head}_h)W^O$$

Sub-problem 5

Feed-Forward Network

$$\text{FFN}(x) = \text{ReLU}(xW_1 + b_1)W_2 + b_2$$

Applied to each position independently

A major locus where

Facts, patterns, associations — encoded in the weights
